Table of contents

## What you'll find in downloads folder
- .zip file with addon version that supports multiple planet creation with atmosphere (but without rings yet)
- Earth, Moon, Mars.blend files that has a planet with right textures(for this reason they are big files) and accurate parameter settings.
- List with texture links below in case you want to build planet yourself. (larger or smaller depending on the need)
- experimental ring system in a separate blend file (Saturn.blend).

## How to use it
- install addon physical-celestial-objects-0.1.0.zip
- To see Earth, Moon and Mars open according blendfiles.
- To see how does rings and atmosphere look like - open Saturn.blend (Currently UI controls are not available. This comes in upcoming weeks)


+ For better visual experience Uncheck 'viewport denoising' under render settings/sampling tab. For our blend files its already done.

Also Feel free to experiment with parameters. A real Documentation comming soon.
